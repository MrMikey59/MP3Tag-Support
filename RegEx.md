# MP3Tag Regular Expressions  

| Action | RegEx |  
| --- | --- |  
| the first three characters | ^.{3} |  
| the last three characters | .{3}$ |  
| remove leading spaces | ^\s+ |  
| remove all trailing spaces | \s+$ |  
| cut double space to single | \s{2,} |  
| remove hyphens with space | \s*-\s* |  
| Remove leading numbers | ^\s*\d |
